Test Instructions

Keyboard Controls:

  • Press U - Should rotate top layer clockwise
  • Press Shift+U - Should rotate top layer counter-clockwise
  • Press D - Should rotate bottom layer clockwise
  • Press L - Should rotate left layer clockwise
  • Press R - Should rotate right layer clockwise
  • Press F - Should rotate front layer clockwise
  • Press B - Should rotate back layer clockwise

Button Controls:

  • Click any move button (U, U', U2, etc.) - Should animate the corresponding layer
  • Click "New Scramble" - Should generate and execute a random scramble
  • Click "Reset" - Should return cube to solved state

Expected Behavior:

  • ✓ Smooth 3D rotation animations
  • ✓ Correct layer rotations (90 degrees)
  • ✓ Stickers maintain proper orientation after rotation
  • ✓ Multiple moves queue properly
  • ✓ OrbitControls allow viewing from different angles

Common Test Sequences

Try these sequences in the input field:

  • R U R' U' - Should return to solved state after 6 repetitions
  • F2 U2 F2 U2 F2 U2 - Should create a checkerboard pattern
  • R U2 R' U' R U' R' - Basic algorithm test
